# Course Management System - Backend

## 📌 Overview
This is the **backend** of the Course Management System built using **Java Spring Boot**.  
It provides REST APIs to manage courses and users.

This backend handles:
- Course creation
- Course retrieval
- Course update
- Course deletion
- API communication with frontend

---

## 🚀 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Maven
- MySQL
- REST API

---

## ✨ Features
- Add new course
- View all courses
- Update course details
- Delete course
- REST API integration
- Backend logic for course management
